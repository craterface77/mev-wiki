A few notes on using the table:

- The links in rows 9–14 (Flashbots writings, EIPs, Ethereum Research) are **not** arxiv — they're primary docs / EIPs. They change more often than papers, but they are the canonical authoritative sources in those topics.
- Paper #4 (CEX-DEX, 2025) is the most recent and most directly aligned with the current state of the market. If you only read one work from tier 2 — read this one.
- Paper #6 (am-AMM) — note that one of the co-authors is Moallemi (same as LVR). This is a deliberate continuation of the line: they first described the problem, then proposed a mechanism to solve it. A logical pairing for sequential reading.
- If you end up writing a research write-up on your dynamic-fee hook via BEX — the standard citation base is #2 (LVR), #6 (am-AMM as the closest mechanism design cousin), #8 (RVR as a benchmark refinement), #4 (for empirical calibration of the problem's scale).
